# Icon 图标

常用的图标，多用于 [`m-button-v2`](/document/ManagerUI_V2/m-button/readme) 的图标选择。

## `MIconV2Type`

| 属性名 | 类型     | 默认值 | 描述      |
| ------ | -------- | ------ | --------- |
| name   | `string` | —      | icon 名称 |
| tip    | `string` | —      | icon 提示 |
