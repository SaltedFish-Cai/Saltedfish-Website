# Icon 图标

常用的图标，多用于 [`pa-button`](/document/PancakeUI_Doc/button/readme) 的图标选择。

## 基础用法

使用 `name` 选择图标

<demo src="./base.vue"></demo>

## 带提示

使用 `tip` 来定义 icon 的 hover 提示

<demo src="./tip.vue"></demo>

## `SaIconType`

| 属性名 | 描述      | 类型     | 默认值 |
| ------ | --------- | -------- | ------ |
| name   | icon 名称 | `string` | —      |
| tip    | icon 提示 | `string` | —      |

## Icon 选择

全 `Icon` 展示直接点击即可获取 `pa-icon` 所需的 `name` 值
